# Hospital Resource Utilization Dashboard

## Problem Statement
Hospitals manage patients, beds, doctors, and resources every day.
Without proper analytics, it becomes difficult for hospital administrators
to track performance and make quick decisions.

This project aims to create a simple analytics dashboard to monitor
hospital resource utilization and patient outcomes.


## Solution Overview
This project provides an interactive dashboard that shows important
hospital KPIs such as bed occupancy, average length of stay,
doctor utilization, readmission rate, and cost per patient.

## Key KPIs
- Average Length of Stay (ALOS)
- Bed Occupancy Rate
- Doctor Utilization
- Readmission Rate
- Emergency vs Scheduled Admissions
- Cost per Patient


## Dataset
This project uses **synthetic hospital data** created only for learning
and demonstration purposes. No real patient data is used.


## Dashboard Features
- Overall hospital performance overview
- Department-wise analysis
- Patient outcome analysis
- Resource utilization insights
- Simple interactive filters


## Tools Used
- Power BI
- PostgreSQL
- SQL
- Python (optional)
- CSV datasets
